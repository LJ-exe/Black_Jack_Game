# 🃏 Blackjack Game (Python)

A simple command-line Blackjack game built with Python. The player competes against a dealer with basic betting mechanics and standard Blackjack rules.

---

## 🎮 Features
- Random card deck generation
- Classic Blackjack rules (hit or stay)
- Dealer plays automatically (hits until 17)
- Ace handling (11 or 1 adjustment)
- Chip-based betting system
- Replay system

---

## 🧠 How to Play
- Start with 100 chips
- Place your bet each round
- Try to get as close to 21 as possible without going over
- Beat the dealer to win chips
- If you reach 0 chips → game over

---

## 🚀 How to Run

```bash
python BlackJack_21.py
